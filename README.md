## Installation

Clone the sources.

```
git clone https://github.com/korczis/microscratch.git
```

Enter the directory.

```
cd microscratch
```

Install npm modules.

```
npm install
```

Run tests.

```
npm test
```

Run server.

```
node app.js
```
